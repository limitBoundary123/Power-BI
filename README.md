## 项目目标
围绕电商商品经营核心问题搭建可交互看板，用于回答：
- 今年/某条商品线（男装/女装）GMV 与订单规模如何？
- GMV Top 商品与贡献占比是多少？是否头部集中？
- 品类结构（GMV占比）如何？哪些品类是高规模/高毛利的“优质品类”？
- 通过明细表联动下钻到商品与品牌，快速定位贡献来源
